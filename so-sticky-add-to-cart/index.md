# Introduction

XB Wishlist is a powerful Shopify app that enhances your store's functionality by allowing customers to create and manage wishlists. This feature can significantly improve user experience and potentially increase sales by enabling customers to save products for future purchase.

## Key Features

With XB Wishlist, you can:

- **Easy Addition**: Provide easy addition of wishlist items to the cart
- **Guest Wishlist**: Allow customers to create wishlists without logging in
- **Mobile Responsive**: Ensure a seamless shopping experience across all devices
- **Multiple Languages**: Support multiple languages for a more personalized shopping experience
- **Wishlisting Product Variants**: Allow customers to wishlist product variants
- **Social Sharing**: Enable sharing of wishlists via social media or email
- **Conversion Analytics**: Monitor and analyze customer wishlist activity to gain valuable insights into product popularity and create targeted marketing campaigns
- **Customization**: Customize the appearance of wishlist buttons and pages
- **Custom CSS & JS**: Customize the appearance of wishlist buttons and pages with CSS and JavaScript

:::tip Why use XB Wishlist?
XB Wishlist helps increase customer engagement, encourages return visits, and provides valuable insights into customer preferences, potentially boosting your store's conversion rates.
:::
