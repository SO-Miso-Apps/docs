## Key Features

:::info Feature Highlights
- **Smart Variant Sync** - Real-time synchronization between sticky cart and main form
- **Live Preview** - See changes instantly with visual customizer
- **25+ Preset Themes** - One-click professional designs
- **Wishlist Integration** - Connect with popular wishlist apps (XB Wishlist)
- **8 Button Effects** - Pulse, Glow, Ripple, Scale, Shine, Bounce, Slide, None
- **Mobile-First** - Perfect responsive design with smart triggers
- **Multi-Language** - Full translation support
:::
