# Installation

Install and activate Sticky Add-to-Cart in under 5 minutes.

## Prerequisites

- Active Shopify store with admin access
- Compatible theme (works with all Online Store 2.0 themes)

## Installation Steps

### 1. Install from Shopify App Store

1. Open [Shopify App Store](https://apps.shopify.com)
2. Search for "Sticky Add to Cart"
3. Click **Add app** → **Install app**

**Permissions required:**
- Read products and variants
- Access store theme
- Store configuration data

### 2. Enable Theme Extension

The app block should auto-activate, but verify:

1. Go to **Online Store → Themes**
2. Click **Customize** on your active theme
3. Open any product page template
4. Check for **Sticky Add to Cart** in App blocks
5. If not visible, add it from the Apps section
6. **Save**

:::tip Quick Check
Visit any product page and scroll down. The sticky cart should appear at the bottom.
:::

### 3. Configure Settings

1. Return to app dashboard
2. Click **Open Customizer**
3. Choose a preset or customize colors/fonts
4. **Save** → **Deploy**

Done! Your sticky cart is now live.

## Verify It Works

1. Visit a product page on your store
2. Scroll down
3. Sticky cart appears at bottom
4. Test variant selection and add to cart

:::warning Not Seeing Changes?
Clear browser cache or use incognito mode.
:::

## Troubleshooting

**App not appearing?**
- Verify theme extension is enabled (Step 2)
- Clear cache and refresh
- Check you're on a product page (not homepage)

**Permission errors?**
- Ensure admin privileges
- Log out and back in
