# Installation

Get up and running in under 5 minutes.

## Prerequisites

- Active Shopify store with admin access
- Online Store 2.0 theme (most modern themes)

## Step 1: Install from App Store

1. Visit [Shopify App Store](https://apps.shopify.com)
2. Search for "SO: Sticky Add to Cart"
3. Click **Add app** â†’ **Install**

## Step 2: Enable Theme Extension

The app block should auto-activate. Verify:

1. **Online Store** â†’ **Themes** â†’ **Customize**

5. **Save**

:::tip Quick Check
Visit a product page and scroll - sticky cart should appear at bottom.
:::
