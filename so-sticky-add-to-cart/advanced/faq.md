# FAQ

Quick answers to common questions.

## General

### Does it work with all themes?

**Yes!** Works with all Online Store 2.0 themes. Older themes need updating first.

### Will it slow my store?

**No.** Lightweight (~50KB), async loading, cached after first load. Minimal impact.

### Works with other apps?

**Yes.** Designed to coexist with other Shopify apps. Contact support if conflicts occur.

### Works on mobile?

**Absolutely!** Mobile-optimized and responsive. Most customers use it on mobile.

### Can I disable on certain pages?

Shows only on product pages. Can disable on desktop via General Settings.

## General Questions

### Does this work with all Shopify themes?

**Yes!** The app uses Shopify's Theme App Extensions, which work with all modern themes (Online Store 2.0). If you have an older theme, updating it will enable compatibility.

### Will this slow down my store?

**No.** The sticky cart is optimized for performance:
- Lightweight code (~50KB)
- Async loading
- Cached after first load
- Minimal impact on page speed

### Can I use it with other apps?

**Yes.** The sticky cart is designed to work alongside other Shopify apps. However, if you experience conflicts with other cart-related apps, contact support.

### Does it work on mobile?

**Absolutely!** The sticky cart is mobile-optimized and responsive. In fact, most customers use it primarily on mobile devices where it provides the most value.

### Can I disable it on certain pages?

The sticky cart appears only on product pages by default. You can disable it on desktop (keeping mobile only) through the General Settings tab.

## Features

### What's the difference between collapsed and expanded?

**Collapsed:**
- Minimal bar showing basics
- Expands when clicked
- Clean, unobtrusive
- Best for simple products

**Expanded:**
- Full panel always visible
- All options shown immediately
- Faster to purchase

### What does the Buy Now button do?

**Yes!** Customers can select all product options (size, color, etc.) directly in the sticky cart without scrolling to the top of the page.

**Yes!** Sale prices are automatically detected and highlighted. You can customize the sale price color to make savings stand out.

### Does it work with product images?

**Yes!** The sticky cart shows the product's featured image. It updates automatically when customers select different variants (if you have variant images set up).
