# Preset Themes

One-click professional themes with coordinated colors, typography, and spacing.

## Quick Apply

1. Customizer → **Presets** tab
2. Browse 25+ themes
3. Click **Apply**
4. Customize (optional)
5. **Save & Deploy**

## Customizing After Apply

**Common tweaks:**

```
Match brand → Colors → Primary BG → Your color
Adjust roundness → Spacing → Border Radius
Change font → Typography → Font Family
```

## Preset Details

### Modern Bold
- Colors: Black (#000), Red sale
- Font: Montserrat, bold
- Radius: 8px
- Style: High contrast, confident

### Minimal Clean
- Colors: Gray (#757575), muted
- Font: Inter, medium
- Radius: 4px
- Style: Subtle, refined

### Luxe
- Colors: Deep brown (#5D4037), burgundy
- Font: Playfair Display
- Radius: 0px (sharp)
- Style: Premium, elegant
